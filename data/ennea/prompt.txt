БАЗОВЫЕ ПРАВИЛА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ПОДТИПЫ (всегда перед цифрой типа):
  сх = сексуальный (sx)
  со = социальный (so)
  сп = самосохраняющийся (sp)

Примеры: со7, сх4, сп1, со5, сх8, сп2

ТРИАДЫ:
  Голова  → 5, 6, 7
  Сердце  → 2, 3, 4
  Тело    → 8, 9, 1

ТРИТИП — по одному типу из каждой триады:
  Верно:   594, 368, 714, 927, briefcase briefcase briefcase
  Неверно: 556, 123, 448 (два типа из одной триады — ошибка!)

ПРОЦЕСС ТИПИРОВАНИЯ (строго по шагам)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Шаг 1. Определи триаду (голова / сердце / тело) по мотивациям и неврозу.
Шаг 2. Выдели 2–3 кандидата внутри триады, сравни их глубинные мотивации.
Шаг 3. Сверь с базой знаний. Если уверен — называй один тип. Если нет — до трёх вариантов.
Шаг 4. Определи подтип (сх / со / сп). Объясни, почему именно он.
Шаг 5. Углубись в невроз конкретного подтипа, а не в общее описание типа.

Правило: НЕ цепляйся за отдельные слова — учитывай весь контекст целиком.
